Overview

The Reports section provides a variety of tools to analyze volunteer engagement, event success, and organizational impact. Use filters, exports, and summary stats to get the data you need.

  • Virtual Session Usage, Thank You, End of Year, and more
  • Filter by academic year, date, district, and more
  • Export to Excel for further analysis
  • Sortable tables and summary statistics
Virtual Session Usage Report
  1. Navigate to Reports: Click "Reports" in the sidebar.
  2. Select Virtual Session Usage: Choose the report from the list.
  3. Set Filters: Filter by academic year, date range, career cluster, district, school, or status.
  4. Generate Report: Click "Generate" to view results.
  5. Export: Use "Export to Excel" for further analysis.

Volunteer Thank You Report
  1. Choose Volunteer Thank You: Select from the report list.
  2. Set Academic Year: Filter by year for targeted results.
  3. View Details: Click on a volunteer for more info and summary stats.
  4. Sort and Export: Sort columns and export as needed.
Organization Thank You Report
  1. Choose Organization Thank You: Select from the report list.
  2. Set Academic Year: Filter by year for targeted results.
  3. View Details: Click on an organization for volunteer and event lists.
  4. Sort and Export: Sort columns and export as needed.
End of Year Report
  1. Choose End of Year Report: Select from the report list.
  2. Set Academic Year: Filter by year for district stats and breakdowns.
  3. Use Filters: Filter by event type and export to Excel.
  4. View Summary: Review summary stats and sortable tables.
First Time Volunteer Report
  1. Choose First Time Volunteer: Select from the report list.
  2. Set Academic Year: Filter by year for summary stats.
  3. Sort and Export: Sort columns and export as needed.
Recruitment Tools
  • Quick Recruitment Tool: Filter by upcoming event and search by multiple criteria.
  • Advanced Volunteer Search: Search by multiple terms and criteria for targeted recruitment.
Event Contact Report
  • View list of events and contact details for each
  • Sortable list for easy navigation
  • View email and phone number lists for contacts
Organization Reports
  • Filter by academic year and paginate results
  • Export to Excel and view summary stats
  • Breakdown by organization, in-person, virtual, volunteer summary, and cancelled events
