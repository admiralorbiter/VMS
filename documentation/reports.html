Virtual Session Usage Report
About This Report
  • This report displays Virtual Sessions (Connector Sessions) imported from the Connector Spreadsheets maintained by Brittany.
  • Each academic year is tied to a specific Google Sheet. Data is imported and not live-edited in the app.
  • Session data includes teacher registrations, school/district info, and session details as recorded in the spreadsheet.
  • Filters and summary stats are based on the imported data for the selected year.
Spreadsheet Management: Google Sheet/Virtual Spreadsheet Management (coming soon)
  1. Navigate to Reports: Click "Reports" in the sidebar.
  2. Select Virtual Session Usage: Choose the report from the list.
  3. Set Filters: Filter by academic year, date range, career cluster, district, school, or status.
  4. Generate Report: Click "Generate" to view results.
  5. Export: Use "Export to Excel" for further analysis.

District Detail View

About the District Detail View
  • This view provides a detailed, month-by-month breakdown of virtual session usage for a single district.
  • Access this view by clicking a district card in the main Virtual Session Usage Report.
  • Each month is collapsible and shows compact stats (sessions, educators, schools) and a detailed table of all sessions for that month.
  • Session tables include: Date, Session Title (with links), Schools, Teachers, and Presenter.
  • Filter by virtual year and date range at the top of the page.
  • Breakdowns by school and teacher are shown above the monthly breakdowns.
  • Designed for easy navigation, comparison, and export of district-level virtual session data.
How to Use:
  1. From the main Virtual Session Usage Report, click a district card to open the detail view.
  2. Use the filters at the top to select year or date range.
  3. Expand/collapse months to focus on specific periods.
  4. Review session details, school and teacher breakdowns, and export as needed.
Volunteer Thank You Report
About This Report
  • This report provides a breakdown of volunteer engagement by academic year. You can filter by host (All Events or PREPKC Events Only) and export the data for further analysis.
  • For each volunteer, the report tracks:
    • Presenter Name
    • Total Hours volunteered
    • Total Events participated in
    • Organization affiliation
  • Clicking a volunteer's name opens a detail view showing:
    • All events the volunteer participated in for the selected year
    • Event date, title, type, hours, school, and district for each event
    • Totals for hours and events for the year
üí° Pro Tip

This report keeps a running total throughout the year, so you can monitor volunteer engagement as the year progresses.

  1. Choose Volunteer Thank You: Select from the report list.
  2. Set Academic Year: Filter by year for targeted results.
  3. View Details: Click on a volunteer for a detailed breakdown of their participation.
  4. Sort and Export: Sort columns and export as needed.

Recruitment Tools
About Recruitment Tools

Recruitment Tools help you match volunteers with upcoming events and opportunities. Choose the tool that best fits your workflow:

  • Quick Recruitment Tool:
    Best for when you already have a few volunteers in mind or want to quickly look up and match people to specific upcoming events.
    • View upcoming unfilled events and filter by event type
    • Search volunteers by name, skills, or organization
    • See volunteer skills, last activity, and contact info next to event needs
    • Direct links to volunteer profiles and Salesforce records
    • Fast filtering for last-minute or targeted recruitment
  • Advanced Volunteer Search:
    Best for exploring your entire volunteer pool using powerful filters and analyzing volunteer history, skills, and engagement.
    • Search by name, skills, organization, past events, and more
    • Flexible search modes: "Wide" (any term) or "Narrow" (all terms)
    • Detailed results table with sorting and pagination
    • Profile and Salesforce links for each volunteer
    • Ideal for planning campaigns or discovering new volunteers
üí° Pro Tip

Use the Quick Recruitment Tool for fast, targeted outreach when you know who you want, and the Advanced Volunteer Search for deeper analysis, discovery, and strategic recruitment.

  1. Choose a Tool: Select either Quick Recruitment or Advanced Volunteer Search from the recruitment tools page.
  2. Set Filters & Search: Use the available filters and search options to find the right volunteers for your needs.
  3. Match & Contact: Review volunteer details, then reach out or assign as needed.

Attendance Details Report
About This Report
  • This report allows you to view and edit detailed attendance data for each event in a given academic year.
  • Track the number of students reached, classrooms/tables used, and students per volunteer‚Äîkey metrics for understanding event impact.
  • Filter by academic year and event type using interactive checkboxes and dropdowns.
  • Edit attendance details directly in the report to keep data accurate and up to date.

  3. Review & Edit Details: View and update the number of students, tables, and other attendance details for each event.
  4. Monitor Impact: Use the students per volunteer metric to assess event effectiveness and reach.
