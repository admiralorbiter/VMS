# Public Volunteer Signup

This guide explains the public-facing signup flow for context.

## 1. Volunteer Hub

Volunteers access opportunities at `prepkc.org/volunteer.html`.
-   **No Account Needed**: Volunteers can sign up without a login.
-   **Events Listed**: In-Person, Virtual (if public), and other opportunities.

## 2. The Signup Process

1.  Volunteer clicks **"Sign Up"** on an event.
2.  Fills out the form:
    -   **Required**: Name, Email, Organization.
    -   **Demographics**: Skills, Education, etc. (used for reporting).
3.  **Submission**:
    -   System checks for errors.
    -   Creates a **Participation Record**.
    -   Sends **Confirmation Email** & **Calendar Invite**.

## 3. Post-Signup

-   **Confirmation**: Volunteer receives an email immediately.
-   **Reminders**: System sends reminders closer to the event.
-   **Updates**: If event details change (in Salesforce), updates are synced and notifications sent.
