Calendar

View, filter, and manage all events in a calendar view.

Overview

The Calendar provides a visual overview of all scheduled events, making it easy to see upcoming activities, filter by type, and quickly access event details.

Viewing the Calendar
  1. Navigate to the Calendar section from the sidebar.
  2. Browse by month, week, or day using the navigation controls.
  3. Click on any event to view its details.
ℹ️ Info

Color coding helps distinguish event types at a glance.

Adding & Editing Events
  1. Click the "Add Event" button or select a date on the calendar.
  2. Fill in event details (name, date, time, location, description).
  3. Save to add the event to the calendar.
  4. To edit, click on an event and select "Edit".
💡 Tip

Recurring events can be set up for regular meetings or sessions.

Filtering & Navigation
  • Filter events by type, school, or volunteer.
  • Use search to quickly find specific events.
  • Switch between month, week, and day views for different perspectives.
ℹ️ Info

Use filters to declutter the calendar and focus on relevant events.

💡 Best Practice

Keep your calendar up to date and use filters to manage large numbers of events efficiently.
