Managing Volunteers

Everything you need to know about volunteer management

Adding a New Volunteer ▼
  1. Navigate to Volunteer Directory
    Click "Volunteers" in the main navigation menu
  2. Click "Add New Volunteer"
    Look for the blue "Add" button at the top of the page
  3. Fill in Required Information
    Complete the personal and contact information fields
  4. Add Professional Details
    Include organization, role, and relevant skills
  5. Save the Record
    Click "Save" to add the volunteer to your database
ℹ️ Required Fields

Name and email address are required. All other fields are optional but recommended for better matching with events.

Importing Volunteers from Spreadsheet ▼
  1. Prepare Your Spreadsheet
    Ensure your Excel or CSV file has columns for name, email, and other volunteer data
  2. Access Import Function
    In Volunteer Directory, click "Import" button
  3. Upload Your File
    Select your spreadsheet file and click "Upload"
  4. Map Columns
    Match your spreadsheet columns to system fields
  5. Review and Import
    Check the preview and click "Import Data"
⚠️ Import Tips

Always backup your data before importing. Duplicate email addresses will be flagged for review.

Searching and Filtering Volunteers ▼

Quick Search Options:

  • Name: Type any part of first or last name
  • Organization: Find volunteers from specific companies
  • Skills: Search by volunteer expertise
  • Email: Locate specific volunteer by email
  • Local Status: Filter by volunteer availability

Advanced Search:

Click "Advanced Search" to combine multiple criteria and create complex queries for targeted volunteer recruitment.

Managing Events

Create, organize, and track your volunteer events

Creating a New Event ▼
  1. Go to Event Directory
    Click "Events" in the main navigation
  2. Click "Add New Event"
    Find the "Add" button at the top of the events list
  3. Enter Basic Information
    Add event name, type, dates, and location
  4. Set Participation Details
    Define how many volunteers and students are needed
  5. Add Skills Requirements
    Specify what skills volunteers should have
  6. Write Descriptions
    Create compelling descriptions for volunteers and students
  7. Save and Publish
    Save your event to make it available for volunteer signup
Managing Event Participation ▼

Adding Volunteers to Events:

  1. Open the event from the Event Directory
  2. Go to the "Volunteer Participation" section
  3. Click "Add Volunteer" and search for volunteers
  4. Select volunteers and confirm their participation
💡 Recruitment Tip

Use the Recruitment Tools to find volunteers with specific skills that match your event needs.

Calendar & Schedule

View and manage upcoming events

Using the Calendar View ▼

Calendar Views:

  • Month View: See all events for the current month
  • Week View: Detailed view of a single week
  • Day View: Hour-by-hour schedule for one day
  • List View: Simple list of upcoming events

Quick Actions:

  • Click on any event to see details
  • Use the "More" link to go directly to the event page
