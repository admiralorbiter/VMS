Common Questions

How do I change my password?
  1. Go to the Admin Dashboard (requires appropriate permissions).
  2. Find the Change Password section.
  3. Enter your new password and confirm it.
  4. Click the Change Password button.

You can find this section in the Admin Dashboard page (admin.html template).

What does the blue cloud button mean?

This button is a quick link to the volunteer's record in Salesforce. Clicking it will open the corresponding Salesforce contact page in a new tab. You may see this button on volunteer profile pages and other areas where Salesforce integration is available.

Salesforce Link

If you have access, clicking this button will take you directly to the Salesforce record for more details or updates.
