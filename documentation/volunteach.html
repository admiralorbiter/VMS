Volunteach - Manage Volunteer Even Sign Ups

The Volunteach microservice acts as a bridge between Salesforce and the Volunteer Sign Up page, syncing event data and enabling flexible event management for PREPKC.

Core Features
  • Sync Events: Hit the Sync button to pull the latest events from Salesforce into the local database.
  • Event Management:
    • Toggle events to display on the public sign-up page.
    • Assign events to specific districts for district-specific sign-up pages.
    • Add notes to events (displayed on the sign-up page).
  • Automatic Cleanup:
    • Events in the database not present in the latest Salesforce sync are flagged for deletion.
    • Events with a start date in the past are automatically deleted.
How it Works
  1. Click Sync to fetch events from Salesforce.
  2. Review and manage events in the dashboard.
  3. Toggle visibility, assign districts, and add notes as needed.
  4. Changes are reflected on the public sign-up pages and district-specific pages.
Best Practices & Tips
  • Always sync before making changes to ensure you have the latest event data.
  • Use the district assignment feature to target events to the right audience.
  • Keep event notes clear and concise for volunteers.
  • Regularly review flagged events to keep your sign-up pages up to date.
