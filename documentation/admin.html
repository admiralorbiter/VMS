Admin

Manage users, imports, bug reports, and advanced system tools in the VMS.

Overview

The Admin section provides tools for user management, data import, bug reporting, and advanced system configuration. Only users with admin privileges can access these features.

  • User management (create, edit, delete, change password)
  • Import and automation tools
  • Bug report management
  • Data purge and cache refresh
User Management
  1. Create New User: Click "Add User" and fill in the required information.
  2. Edit User: Click the "Edit" button next to a user to update their details.
  3. Change Password: Use the "Change Password" option for any user.
  4. Delete User: (Admin only) Use the "Delete" button to remove a user.
ℹ️ Note

Deleting a user is permanent. Make sure to review user roles and permissions regularly.

Import Process & Automation
  • Automation: Set up automated imports for regular data updates.
  • Sequential Data Import: Use "One Click" to import all required data in sequence.
  • Salesforce Imports: Import data directly from Salesforce.
  • Purge Data: Remove all volunteer or event data as needed.
  • Refresh Caches: Update all cached data for the latest information.
⚠️ Caution

Purge and refresh actions are irreversible. Always backup your data before performing these operations.

Bug Reports
  1. Submit Bug: Use the bug icon in the sidebar or the admin panel to submit a new bug report.
  2. View Reports: See active and resolved bug reports in the admin panel.
  3. Resolve or Delete: Mark bugs as resolved or delete them as needed.
💡 Tip

Encourage users to submit detailed bug reports for faster resolution.

💡 Best Practice

Regularly review user accounts, automate imports where possible, and keep your system up to date for best performance.
